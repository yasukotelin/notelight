## Install

If you use any vim plugin manager, refer to the following.

**vim-plug**

```vim
Plug 'yasukotelin/notelight'
```

**dein.vim**

```vim
call dein#add('yasukotelin/notelight')
```

Finally, Write to your `.vimrc` or `init.vim`.

```vimrc
colorscheme shirotelin
```
